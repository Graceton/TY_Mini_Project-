Optivox | User Manual

Welcome to Optivox, your AI-powered companion for a more accessible digital world.

🚀 Getting Started

  • Download & Extract
    Download the latest Optivox.zip. Right-click the file and select "Extract All..." to a folder of your choice.
  • Identify the App
    Open the extracted folder and look for GUI.exe (or GUI.py if you are a developer).
  • First Launch (Security)
    Windows might show a "Windows protected your PC" prompt. Click on "More info" and then "Run anyway". This is required for accessibility tools to interact with your screen.
  • System Tray Integration
    Optivox starts in the background. Look for the Optivox icon in your System Tray (bottom-right of your taskbar). Right-click it to quickly open settings or exit.

🎨 Understanding Color Coding

Optivox uses a color-coded navigation system to help you quickly identify features at a glance.

🔵 Magnification (Blue)
🟢 Text Reader (Green)
🔴 Voice Assistant (Red)
🟣 Settings (Purple)
🌑 Exit Menu (Dark Blue)

⌨️ Keyboard Shortcuts

Master these shortcuts to control Optivox without needing to see the screen clearly.

🛠️ Core Controls

Open Control Panel: Win + O
Start Magnifier: Win + =
Start Text Reader: Ctrl + Shift + Alt + R
Stop All Features: Win + -

🔍 Magnifier Control

Zoom In: Ctrl + +
Zoom Out: Ctrl + -
Reset Zoom: Esc

🎙️ Voice Assistant

Enable "Hands-Free" mode in settings, then say "Hey Optivox" to wake the assistant. You can then say:

"Start Magnifier"
"Open Reader"
"Magnify the screen"

🖥️ Interface Navigation

Cycle Buttons: Tab
Select Feature: Enter
Switch Window: Alt + Tab
