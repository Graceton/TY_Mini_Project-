📖 Official User Manual

🚀 First-Time Setup

1. Extract Download: Right-click the .zip and select "Extract All".

2. Run GUI: Open the folder and run GUI.exe.

3. Security Prompt: If Windows warns you, click "More info" and "Run anyway".

⌨️ Keyboard Shortcuts

  • Win + O: Control Panel
  • Win + =: Start Magnifier
  • Win + -: Stop All Features
  • Ctrl + Shift + Alt + R: Reader
  • Ctrl + / -: Adjust Zoom
  • Esc: Reset Zoom

🎙️ Voice Assistant

Say "Hey Optivox" to wake the assistant, then follow up with commands like "Start Magnifier" or "Open Reader".

🎨 Navigation Color Scheme

Blue: Zoom Green: Reader Red: Voice Purple: Settings
